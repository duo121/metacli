# @duo121/metacli-core

Core JSON, error, CLI spec, and doctor helpers for AI-native CLIs.

## Install

```bash
npm install @duo121/metacli-core
```

## Exports

- `CLIError`
- `toErrorPayload`
- `createJsonText`
- `writeJson`
- `createCliSpec`
- `createDoctorCheck`
- `createDoctorReport`

See the root project for architecture and package layering:
https://github.com/duo121/metacli
