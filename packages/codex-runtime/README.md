# @duo121/metacli-codex-runtime

Codex session helpers built on top of the metacli terminal runtime.

## Install

```bash
npm install @duo121/metacli-core @duo121/metacli-terminal-runtime @duo121/metacli-codex-runtime
```

## Exports

- `attachCodexSession`
- `launchCodexSession`
- `sendCodexPrompt`
- `submitCodexPrompt`
- `captureCodexSession`

See the root project for architecture and examples:
https://github.com/duo121/metacli
