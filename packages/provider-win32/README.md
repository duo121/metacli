# @duo121/metacli-provider-win32

Windows providers for Windows Terminal and Command Prompt.

## Install

```bash
npm install @duo121/metacli-core @duo121/metacli-terminal-runtime @duo121/metacli-provider-win32
```

## Includes

- Windows Terminal provider
- Command Prompt provider
- PowerShell JSON runner
- UI Automation helpers

See the root project for usage examples and capability notes:
https://github.com/duo121/metacli
